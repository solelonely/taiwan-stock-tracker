# 台股走勢查詢系統

一個簡潔美觀的台灣股市即時查詢網頁應用程式

## 功能特色

- 📈 即時股票價格查詢
- 📊 6個月歷史走勢圖表
- 💹 漲跌幅與成交量顯示
- 📱 響應式設計，支援手機瀏覽
- 🎨 現代化 UI 設計

## 使用方式

1. 輸入台股代碼（例如：2330、2317、2454）
2. 點擊查詢按鈕或按 Enter
3. 查看股票資訊與走勢圖

## 常用股票代碼

- 2330 - 台積電
- 2317 - 鴻海
- 2454 - 聯發科
- 2412 - 中華電
- 2882 - 國泰金

## 技術架構

- 純 HTML/CSS/JavaScript，無需後端
- 使用 Yahoo Finance API 取得股票資料
- Chart.js 繪製走勢圖表
- 響應式設計適配各種裝置
